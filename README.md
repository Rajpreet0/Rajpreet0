### 🚀 What I'm building right now

- **Kumasy** — my B2B2C SaaS, built from zero to product.
- **The Enabler** - a GDPR-friendly PII redaction tool
- **Cloud Clipboard** — a cross-platform clipboard sync tool (web + Electron desktop) with device pairing, team sharing, and a Postgres/Prisma backend.

I like taking a product from "just an idea" to something real people can log into — architecture, backend, frontend, and the boring-but-critical parts (auth, CI/CD, deployment) included.

### 🛠️ Featured Projects

| Project | What it is |
|---|---|
| **Since3 Studio** | A client-portal SaaS for a software agency — separate admin & client dashboards, JWT auth with OTP 2FA, encrypted credential vault, and a Node/Express + MongoDB backend. Frontend and backend built and shipped as two coordinated services with their own CI pipelines. |
